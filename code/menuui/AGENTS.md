# Module: menuui

Entry-point guide for this module lives at:
**`documentation/modules/menuui.md`** (from repo root).

It covers the non-mission front-end screens (main hall, pilot select, barracks,
tech room, ready room, options, credits): purpose, the file→screen→game-state
mapping, key concepts, and the configuration tables it parses.

For the engine-wide architecture overview see `documentation/ARCHITECTURE.md`.
For build/test/style conventions see the root `AGENTS.md`.
