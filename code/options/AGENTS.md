# Module: options

Entry-point guide for this module lives at:
**`documentation/modules/options.md`** (from repo root).

It covers the SCP options framework and the in-game Options menu: purpose, key
files, core data structures (`OptionBuilder`/`Option`/`OptionsManager`), major
enums/flags, how to declare an option, persistence/defaults, and the
`default_settings.tbl` it parses.

For the engine-wide architecture overview see `documentation/ARCHITECTURE.md`.
For build/test/style conventions see the root `AGENTS.md`.
