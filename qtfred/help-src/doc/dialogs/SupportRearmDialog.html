Support Options

Opens via Editors › Mission Specs › Support Options.

Controls whether support ships appear in the mission, how much they can repair, whether they can rearm weapons, and how many of each weapon they carry in their rearm pool.

Repair settings

ControlDescription
Allow Support ShipsWhen unchecked, no support ship can appear in this mission regardless of the loadout.
Support Ships Repair HullAllows support ships to repair hull damage in addition to subsystems. Subsystem repair is always available when support ships are allowed.
Hull Repair CeilingMaximum percentage of hull integrity a support ship can restore. Only meaningful when hull repair is enabled.
Subsystem Repair CeilingMaximum percentage of subsystem integrity a support ship can restore.

Rearm options

ControlDescription
Disallow Support RearmPrevents support ships from rearming any weapons. Repair is still permitted if support ships are allowed.
Limit Rearm to Mission PoolRestricts rearming to the quantities defined in the Support Rearm Pool table below. Once a weapon's pool is exhausted, support ships cannot supply more of it. A pool value of -1 means unlimited for that weapon.
Set Rearm Pool from Loadout Pool surplusInitialises each weapon's pool amount from any surplus left over in the player loadout pool at mission start, in addition to any values set manually.
Use Weapon Precedence when pool is emptyWhen a weapon's pool is exhausted, the support ship substitutes a weapon from the precedence list defined in the weapon tables rather than simply refusing to rearm.

Support Rearm Pool

Sets how many of each weapon the support ships can supply across the entire mission. Only active when Limit Rearm to Mission Pool is checked. Each team in a Team vs. Team mission has its own independent pool; use Pool Team to switch between them.

Select a weapon in the list to edit its pool amount, then use the buttons to apply a value. The amount field accepts any integer; use -1 for unlimited.

ControlDescription
Pool TeamSelects which team's pool is shown. In non-Team vs. Team missions only one team entry is present.
Weapon listLists all player-allowed weapons. Select a weapon to target it with the amount buttons.
Pool AmountAmount to apply when clicking Set Amount or Set ALL to Amount.
Set AmountSets the selected weapon's pool to the value in Pool Amount.
Set Unlimited (-1)Sets the selected weapon's pool to -1 (unlimited).
Set ZeroSets the selected weapon's pool to 0, preventing support ships from supplying it entirely.
Set ALL to AmountApplies the Pool Amount value to every weapon in the list at once.
Set ALL to UnlimitedSets every weapon's pool to -1.
Set ALL to ZeroSets every weapon's pool to 0.
